Listagem das principais tags da HTML:

  • <head>

    This tag contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.

  • <style>

    contém informações de estilo para um documento ou uma parte do documento.

  • <title>

    • O elemento title define o título do documento que é exibido na barra de título de um navegador ou na aba de uma página. Ele contém apenas texto; tags dentro do elemento são ignoradas.

  • <body>

    • O elemento body representa o conteúdo de um documento HTML. Só pode haver um body em um documento.

  • <footer>

    • O elemento footer representa um rodapé para seu ancestral mais próximo. Um tipicamente contém informações sobre o autor da seção, dados de copyright ou links para documentos relacionados.

  • <label>

    • O elemento label representa uma legenda para um item em uma interface de usuário. Geralmente usado para formulários.

  • <i>

    • O elemento "i" representa uma forma de diferenciar as palavras deixando-as em italico.

  • <strong>

    • O elemento strong representa uma forma de diferenciar as palavras deixando-as em negrito.

  • <small>

    • O elemento small representa uma forma de diferenciar as palavras deixando-as em fonte menor.

  • <br>

    • O elemento br representa uma forma de quebrar uma linha no meio de um paragrafo.

  • <hr>

    • O elemento hr representa uma forma de fazer uma quebra tematica. Colocando uma linha no meio.

  • <b>

    • O elemento b indica que o conteudo é importante. deixando em negrito.

  • <div>

    • O elemento div é um container genérico para conteudo de fluxo. Não tem efeitos até que seja estilisado no CSS

  • <form>

    • O elemento form representa uma forma de receber informações ao html.

  • <input>

    • O elemento input é uma forma de entrada de dados pelo usuario.

  • <tr>

    • O elemento tr define uma linha de cedulas em uma tabela

  • <td>

    • O elemento td define uma cedula de uma tabela que contem dados

  • <span>

    • O elemento span é um container para frasear conteudo que não representa valor. Muito utilizado para estilizar codigo

  • <img>

    • O elemento img incorpora uma imagem no documento

  • <audio>

    • O elemento audio incorpora um conteudo sonoro no documento.

  • <canvas>

    • O elemento canvas cria um espaço de pixes que podem ser estilizados. Exemplo um quadrado verde

  • <datalist>

    • O elemento datalist contém um conjunto de elementos que representam as opções permitidas ou recomendadas
    disponíveis para escolha em outros controles. Lista de escolha de sabor de sorvete

  • <details>

    • O elemento details cria um widget de divulgação no qual as informações são visíveis somente quando o widget é alternado para um estado "aberto".

  • <embed>

    • O elemento embed incorpora conteúdo externo no ponto especificado no documento. Esse conteúdo é fornecido por um aplicativo externo ou outra fonte de conteúdo interativo, como um plug-in de navegador.

  • <ul>

    • O elemento ul representa uma lista não ordenada de itens

  • <ol>

    • O elemento ol representa uma lista ordenada de itens geralmente numerada.

  • <li>

    • O elemento li representa um item de uma lista que esteja dentro do pai ul ou ol/p>
